1) Install Python 3
2) Set PYTHONPATH 

3) Install [Snakemake](https://snakemake.readthedocs.io/en/stable/getting_started/installation.html)

4) Make sure R >= 3.6 is installed and that it is on your PATH
5) Check for the following R libraries: 
     - cowplot
     - funr
     - ggplot2
     - gridExtra
     - logger
     - parallel
     - randtoolbox
     - rgeos
     - R.utils
     - SearchTrees
     - sp
     - tidyverse
     - xlsx
     - yaml
